How it works

AI Stack is a thin, smart launcher. It starts both services, wires them together, and stays out of your way.

Double-click the desktop icon or run docker compose up -d on your VPS.

Desktop App

Install on Windows

A native installer with a desktop shortcut. Download once, use forever.

1

Install Node.js 20+

AI Stack needs Node.js to run OpenCode and OpenChamber.

Download the LTS version from nodejs.org. Restart your PC after installing. 
node --version

You should see v20.x.x or higher.

2

Install OpenCode & OpenChamber globally

Open Command Prompt or PowerShell and run:

npm install -g opencode-ai @openchamber/web

If you skip this step, AI Stack will prompt you to install on first launch.

3

Download AI Stack Installer

Download AI Stack Setup.exe
SmartScreen may show a blue prompt — click "More info" then "Run anyway". The file is safe.
4

Run the Installer

Double-click AI Stack Setup.exe. In under 30 seconds you'll have:

  • A desktop shortcut icon
  • Start Menu entry
  • Everything ready to launch
5

Add your API Key

Open AI Stack → Settings → Providers → paste your API key.

VPS / Web App

Deploy on a VPS

Run the full stack on any cloud server. Access from your laptop, tablet, or phone.

1

Get a VPS

Any Linux VPS with 1 GB+ RAM works:

DigitalOcean Hetzner Vultr Linode AWS EC2
2

Install Docker

curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER && newgrp docker
3

Download compose file

curl -fsSL https://raw.githubusercontent.com/Akakaui/opencode-desktop-app/main/docker/docker-compose.yml -o docker-compose.yml
4

Start the stack

docker compose up -d
docker compose ps
5

Open in browser

http://<your-vps-ip>:3000
Open port 3000: sudo ufw allow 3000
6

Add a domain & HTTPS

Use Nginx + Let's Encrypt for a clean https://ai.yourdomain.com URL:

sudo apt install nginx certbot python3-certbot-nginx -y

Requirements

  • OS Ubuntu 22.04 / 24.04
  • RAM 1 GB min (2 GB rec.)
  • CPU 1 vCPU+
  • Port 3000 open

FAQ

Common questions

Desktop vs VPS — what's the difference?

The desktop installer bundles everything into a native Windows app. The VPS deployment runs on a cloud server so you can access it from any device.

Does it work on macOS or Linux?

The .exe installer is Windows only. On macOS/Linux, clone the repo, run cd desktop && npm install && npm start. The VPS deployment works on any OS via Docker.

Will updates break anything?

No. AI Stack is a thin wrapper — it just runs opencode serve and openchamber serve. Updates are picked up automatically.

How do I set a password?

Inside OpenChamber: Settings → Security → set a password. AI Stack doesn't control this.

What if ports 3000 or 4095 are in use?

AI Stack detects occupied ports and skips starting that service. Use env vars to change them.

Is my data stored locally?

Yes. Desktop data goes to %APPDATA%/ai-stack-desktop. VPS data goes to a Docker volume on your server.
